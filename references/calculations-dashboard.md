# Calculations And Dashboard

Use these formulas for dashboards and reports. Label results as estimates when inputs are incomplete.

## Budget

```text
monthly_income = take_home_income + average_side_income
total_spending = fixed_costs + variable_costs
savings_amount = monthly_income - total_spending
savings_rate = savings_amount / monthly_income
fixed_cost_ratio = fixed_costs / monthly_income
variable_cost_ratio = variable_costs / monthly_income
```

## Debt

```text
debt_to_income = total_monthly_debt_payments / monthly_income
total_debt_ratio = total_debt / annual_income
```

Use amortization formulas for installment loans. For credit-card or revolving debt, treat rates and payments as approximate unless the statement is available.

## Net Worth

```text
total_assets = cash + deposits + investments + retirement_accounts + real_estate + jeonse_deposit + other_assets
total_liabilities = credit_loans + card_debt + jeonse_loans + mortgages + auto_loans + student_loans + other_debt
net_worth = total_assets - total_liabilities
liquid_asset_ratio = liquid_assets / total_assets
risk_asset_ratio = risk_assets / total_assets
```

## FIRE

```text
annual_expenses = monthly_expenses * 12
required_fire_assets = annual_expenses / withdrawal_rate
fire_gap = required_fire_assets - current_investable_assets
```

## Retirement

```text
annual_retirement_spending_gap = expected_annual_spending - expected_annual_pension_income
required_retirement_capital = annual_retirement_spending_gap / withdrawal_rate
future_value = current_assets * (1 + return_rate) ^ years + monthly_saving * (((1 + monthly_return) ^ months - 1) / monthly_return)
```

## Housing

```text
monthly_payment = principal * monthly_rate * (1 + monthly_rate) ^ months / ((1 + monthly_rate) ^ months - 1)
housing_burden = monthly_housing_cost / monthly_income
loan_payment_burden = monthly_debt_payments / monthly_income
```

## Financial Health Score

Total 100:

- Cash flow and budgeting: 20
- Debt stability: 20
- Emergency fund and protection: 15
- Investment and allocation: 15
- Pension and retirement readiness: 15
- Housing, tax, and system usage: 15

Dashboard should show:

- Score and grade
- Net worth
- Savings rate
- Emergency fund months
- Debt payment burden
- Asset allocation
- Retirement gap
- 90-day actions

Use colors carefully:

- Excellent: green
- Healthy: blue
- Watch: amber
- Weak: orange
- Critical: red

Avoid frightening labels; pair weak areas with actions.
