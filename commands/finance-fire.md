# `/finance fire`

Use for FIRE and financial independence estimates.

## Inputs

- Monthly living expenses
- Current investable net worth
- Monthly saving/investment
- Target withdrawal rate
- Expected return assumptions
- Target lifestyle and housing assumptions

## Analyze

- Annual expenses.
- Required net worth at target withdrawal rate.
- Current gap.
- Estimated time to target under conservative, base, and aggressive scenarios.

Use `scripts/fire_calculator.py` when numeric values are available.

## Output

- FIRE number.
- Scenario table.
- Main sensitivity: living expenses, savings rate, returns, housing.
- 90-day action plan.

Make clear that withdrawal rates and return assumptions are planning assumptions, not guarantees.
