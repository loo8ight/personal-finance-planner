# `/finance`

Use for starting a complete personal finance plan or resuming an existing plan.

## Steps

1. Check whether `1-my-profile.md` exists in the current work folder.
2. If it exists, load existing files and ask what the user wants to update.
3. If it does not exist, confirm Korean residency context and give the privacy notice.
4. Run the 7-round interview from `references/interview-guide.md`, asking 2 to 4 questions at a time.
5. Create or update:
   - `1-my-profile.md`
   - `2-my-budget.md`
   - `3-my-plan.md`
   - `4-my-dashboard.html`
   - `check-ins/`
   - `shareable/`
   - `README.txt`

## Required Output

- Short Korean summary of the current financial situation.
- Assumptions and missing information.
- Priority list for cash flow, debt, emergency fund, investment allocation, pension, housing, tax, and insurance.
- 90-day action plan.
- Reminder that this is educational support, not professional advice.

## First Response Example

```md
좋아. 먼저 전체 재무 상황을 한 번에 다 묻지 않고, 2~3개씩 나눠서 정리할게.

이 Skill은 한국 거주자 기준의 예산, 부채, 투자, 연금, 주거, 세금 체크를 도와주는 개인 재무관리 도구야. 금융·세무·법률 자문은 아니고, 입력값 기반의 교육용 분석과 실행 계획을 제공해.

먼저 이것부터 알려줘.

1. 나이대
2. 직업/소득 형태: 직장인, 프리랜서, 개인사업자, 무직, 학생 등
3. 월 실수령액 또는 월평균 소득
4. 현재 가장 해결하고 싶은 문제: 예산, 카드값, 대출, 투자, 주거, 은퇴, 세금, 기타
```
